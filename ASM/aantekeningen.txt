Data movement
| Command            | Betekenis              | Uitleg                                                                                       |
| ------------------ | ---------------------- | -------------------------------------------------------------------------------------------- |
| `mov dest, src`    | Move                   | Kopieert data van `src` naar `dest` (zonder `src` te verwijderen)                            |
| `lea dest, [addr]` | Load Effective Address | Laadt het **adres** van een geheugenlocatie in `dest` (geen data zelf, maar pointer)         |
| `push reg/mem`     | Push                   | Plaatst een register of waarde op de stack                                                   |
| `pop reg`          | Pop                    | Haalt de bovenste waarde van de stack en zet in register                                     |
| `xor reg, reg`     | Exclusive OR           | Bitwise XOR; vaak gebruikt als snelle manier om een register op 0 te zetten (`xor eax, eax`) |
| `inc reg`          | Increment              | Verhoogt de waarde in een register met 1                                                     |
| `dec reg`          | Decrement              | Verlaagt de waarde in een register met 1                                                     |

Arithmetic & Logic
| Command                       | Betekenis        | Uitleg                                                  |     |
| ----------------------------- | ---------------- | ------------------------------------------------------- | --- |
| `add dest, src`               | Addition         | Dest = Dest + Src                                       |     |
| `sub dest, src`               | Subtract         | Dest = Dest - Src                                       |     |
| `imul dest, src`              | Integer Multiply | Signed integer vermenigvuldiging                        |     |
| `idiv reg/mem`                | Integer Divide   | Signed integer deling; quotient in `rax`, rest in `rdx` |     |
| `and dest, src`               | Bitwise AND      | Dest = Dest & Src                                       |     |
| `or dest, src`                | Bitwise OR       | Dest = Dest                                             | Src |
| `not reg`                     | Bitwise NOT      | Inverteert alle bits in register                        |     |
| `shl dest, n` / `sal dest, n` | Shift Left       | Vermenigvuldigt bits met 2^n                            |     |
| `shr dest, n`                 | Shift Right      | Verdeelt bits door 2^n                                  |     |

Control flow / jumps
| Command                   | Betekenis                    | Uitleg                                                           |
| ------------------------- | ---------------------------- | ---------------------------------------------------------------- |
| `jmp label`               | Jump                         | Onvoorwaardelijk naar label springen                             |
| `je label` / `jz label`   | Jump if equal / zero         | Springt als zero flag (ZF) = 1                                   |
| `jne label` / `jnz label` | Jump if not equal / not zero | Springt als ZF = 0                                               |
| `call label`              | Call                         | Springt naar functie/subroutine en pusht return address op stack |
| `ret`                     | Return                       | Keert terug van functie naar address op stack                    |
| `loop label`              | Loop                         | Vermindert `rcx` en springt als niet nul                         |

Stack / function prologue & epilogue
| Command        | Betekenis         | Uitleg                                    |
| -------------- | ----------------- | ----------------------------------------- |
| `push rbp`     | Save base pointer | Bewaart de oude frame pointer             |
| `mov rbp, rsp` | Set frame pointer | Nieuwe functie‑frame starten              |
| `leave`        | Function epilogue | Equivalent aan `mov rsp, rbp` + `pop rbp` |
| `ret`          | Return            | Springt terug naar caller                 |

System / calls / special
| Command    | Betekenis      | Uitleg                                    |
| ---------- | -------------- | ----------------------------------------- |
| `syscall`  | System call    | Roept Linux kernel op met nummer in `rax` |
| `int 0x80` | Legacy syscall | Oudere Linux syscall interrupt            |
| `nop`      | No operation   | Doet niks; vaak gebruikt voor padding     |
| `hlt`      | Halt           | Stop processor (kernel mode)              |

Registers (x86-64)
| Register     | Betekenis / Gebruik                         |
| ------------ | ------------------------------------------- |
| `rax`        | Accumulator / return value / syscall nummer |
| `rbx`        | Base register (callee-saved)                |
| `rcx`        | Counter / 4e argument / shift count         |
| `rdx`        | Data register / 3e argument                 |
| `rsi`        | Source index / 2e argument                  |
| `rdi`        | Destination index / 1e argument             |
| `rbp`        | Base pointer / frame pointer                |
| `rsp`        | Stack pointer                               |
| `r8` … `r15` | Extra registers / 5e tot 12e argumenten     |

Memory / addressing modes
| Syntax        | Betekenis                         |
| ------------- | --------------------------------- |
| `[rax]`       | Memory op address in `rax`        |
| `[rbp-8]`     | Local variable op stack           |
| `[rel label]` | RIP-relative addressing, PIE-safe |
| `rax`         | Direct register (geen geheugen)   |

rsp (esp, sp)	| stack pointer		| Points to the top address of the stack
